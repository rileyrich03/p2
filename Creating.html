Creating a File

1. Open Visual Studio Code

Launch Visual Studio Code from your applications folder or start menu if you haven't already done so.

2. Create a New File

Click on the "File" menu in the top left corner of the VS Code window, then select "New File" from the dropdown menu. Alternatively, you can use the keyboard shortcut Ctrl + N (Windows/Linux) or Cmd + N (macOS) to create a new file.

3. Write Your Code

In the new file, type or paste code to create a basic HTML document

4. Save Your File

Click on the "File" menu again and select "Save" from the dropdown menu. Choose a location on your computer to save the file, give it a name (e.g., index.html), and click "Save."
