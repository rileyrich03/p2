Install and Create Files with Visual Studio Code

Are you ready to dive into the world of coding? Whether you're a seasoned developer or just starting your journey, having the right tools at your disposal is essential. Visual Studio Code (VS Code) is a powerful, lightweight code editor developed by Microsoft that's perfect for writing, debugging, and deploying your code with ease. In this guide, we'll walk you through the process of installing VS Code and creating your first file to get you up and running in no time.

Materials List

  • PC which fulfills VS code requirements
  • Internet connection
  • Mouse
  • Keyboard
  • Display

VS Code Requirements

  • Operating System:
    • Windows 7 or later
    • macOS 10.10 or later
    • Linux (64-bit only)
  • Processor: 1.6 GHz or faster
  • RAM: 1 GB RAM minimum, 2 GB RAM recommended
  • Hard Disk Space: 200 MB minimum free disk space
  • Display: 1280 x 720 resolution or higher
